emc2.io (emc2.io)¶

This module contains all of the basic I/O functions of EMC^2:

load_mie_file(filename)

Loads the Mie parameters from a file.

load_scat_file(filename, is_radar[, param_type])

Loads ice scattering LUTs from a file.

load_bulk_scat_file(filename[, param_type])

Loads bulk ice or liquid scattering LUTs from a file (by default using the PSD used in the C6 collection).

load_arm_file(filename, **kwargs)

Loads an ARM-compliant netCDF file.
